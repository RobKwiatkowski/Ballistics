Welcome to my first project site

Ballistics program calculates the free fall parameters of an object released from the given height. It includes factorss like aerodynamic drag effect and change of the air parameters with the height. Currently only International Standard Atmosphere (ISA) troposhere is modelled but it will be expanded to the lower stratosphere as well. The program is written in Python.

ISA

International Standard Atmosphere (ISA) is modeled by the approximation functions taken from the NASA webpage. To read more about the ISA go here. Currently model spans to the Tropopause which is as 11000m.

How high it is.

If you are wondering how high it is below are typical operating altitudes of various airplanes:
